# E-Commerce Data Analysis Dashboard

This project is a data analysis and dashboard project built as part of a Dicoding data analysis submission. The project analyzes e-commerce transaction data to generate business insights through exploratory data analysis and an interactive Streamlit dashboard.

## Overview

The goal of this project is to transform raw e-commerce data into meaningful insights that can support business decision-making.

The analysis focuses on customer behavior, sales performance, product categories, payment trends, and delivery performance.

## Objectives

- Clean and prepare e-commerce transaction data
- Perform exploratory data analysis
- Identify sales and customer behavior patterns
- Build an interactive dashboard using Streamlit
- Present business insights in a clear and visual format

## Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- Jupyter Notebook

## How to Run
1. Clone repository
git clone https://github.com/bayufrassetyo/Submission_Proyek_Analisis_Data.git
cd Submission_Proyek_Analisis_Data

2. Install dependencies
pip install -r requirements.txt

3. Run Streamlit dashboard
streamlit run dashboard/dashboard.py
